# Student Performance Analyzer (NumPy Project)

## Description
This project analyzes student performance using NumPy. It demonstrates:
- Random dataset generation
- Broadcasting
- Vectorization (no loops)


## Features
- Generate marks for 100 students
- Apply subject weights using broadcasting
- Compute total and normalized scores
- Assign grades (A, B, C, D, F)
- Identify top students
- Save dataset to CSV

## Technologies Used
- Python
- NumPy

## How to Run

bash
cd src
python main.py
